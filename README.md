## 📌 Workflow Kolaborasi
1. Clone repository ke lokal.  
2. Buat branch sesuai nama masing-masing.  
3. Lakukan perubahan sesuai bagian tugas.  
4. Commit dan push perubahan.  
5. Buat pull request untuk digabungkan ke `main`.
